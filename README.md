# 图片切割转PDF工具

## 功能说明

这个工具可以将竖向合并的图片切割并转换为PDF文件。

文件名格式要求：`xxx-N.png` 或 `xxx-N.jpg`
- `xxx` 是文件名称
- `N` 是数字，表示这个图片实际包含N张图片（竖着合并的）

例如：
- `原密协议-5.png` 表示这个图片包含5张图片，会被切割成5份并生成 `原密协议.pdf`
- `员工手册-33.png` 表示包含33张图片，会被切割成33份并生成 `员工手册.pdf`

## 安装依赖

```bash
pip install -r requirements.txt
```

或者直接安装：

```bash
pip install Pillow
```

## 使用方法

### 方法1：处理当前脚本所在目录

直接运行脚本：

```bash
python cut_images_to_pdf.py
```

### 方法2：指定文件夹路径

```bash
python cut_images_to_pdf.py /path/to/your/folder
```

## 输出

- **PDF文件**：保存在同一个文件夹中，文件名会去掉 `-N` 部分
- **切割后的图片**：每一张切割后的图片都会保存，命名格式为 `xxx-第1页.png`, `xxx-第2页.png` 等
- 原始合并图片文件不会被删除

## 示例

假设有以下文件：
```
原密协议-5.png
发明协议-5.png
劳动合同-12.png
```

运行脚本后会生成：

**PDF文件：**
```
原密协议.pdf  (包含5页)
发明协议.pdf  (包含5页)
劳动合同.pdf  (包含12页)
```

**切割后的图片：**
```
原密协议-第1页.png
原密协议-第2页.png
原密协议-第3页.png
原密协议-第4页.png
原密协议-第5页.png
发明协议-第1页.png
发明协议-第2页.png
... (以此类推)
劳动合同-第1页.png
劳动合同-第2页.png
... (共12张)
```
